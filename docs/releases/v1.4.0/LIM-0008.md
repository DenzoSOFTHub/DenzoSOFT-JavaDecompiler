**Type**: Bug
**Status**: RESOLVED

**Resolved**: 2026-03-25

## Description

Multi-resource try-with-resources shows expanded bytecode form. Accepted as limitation LIM-0008.

## Solution Implemented

Documented as limitation LIM-0008. The bytecode pattern for multi-resource TWR is extremely
complex and varies between JDK versions.
